MS Update - Serviço de Atualização Automática

O MS Update surgiu como resposta a um desafio interno da empresa, relacionado à cansativa tarefa de atualizar o sistema de forma manual. Para superar essa dificuldade, criei um serviço do Windows que interage com uma API externa, responsável por verificar a disponibilidade de novas atualizações.

Por meio dessa solução, conseguimos não apenas automatizar o processo de atualização, mas também otimizar a produtividade da equipe de desenvolvimento, permitindo-lhes concentrar-se na implementação de novas funcionalidades e na resolução ágil de problemas e bugs em ambiente de produção.

Tecnologias utilizadas: C#, .NET, SQL Server, API REST.

Integração com E-commerce

Com o objetivo de integrar o e-commerce de nosso cliente ao nosso ERP, desenvolvemos uma solução que facilita todas as etapas do processo. Agora, nosso cliente pode gerenciar pedidos, realizar vendas através de nosso ERP, emitir notas fiscais eletrônicas, inserir códigos de rastreamento e despachar encomendas, tudo de maneira intuitiva e simplificada.

Os resultados alcançados foram notáveis, a produtividade na entrega de pedidos triplicou e o faturamento do nosso cliente aumentou aproximadamente em 70%.

Tecnologias utilizadas: HTML, CSS, JavaScript (jQuery), C#, .NET, SQL Server, API REST.

Integração com AverbePorto

O processo tradicional de averbação de carga frequentemente resultava em frustrações e atrasos. A comunicação via email era causava problemas de acompanhamento e de forma manual que acabava consumindo um tempo valioso de nosso cliente.

Para isso desenvolvemos uma integração diretamente com o sistema de Averbação de Carga do Porto Seguro, automatizamos e simplificamos todo o processo. O cliente não precisa mais lidar com o envio manual de informações por email. Em vez disso, os dados de carga são transmitidos de forma segura e precisa através da nossa plataforma, eliminando erros de comunicação e acelerando o fluxo de trabalho.

Tecnologias utilizadas: HTML, CSS, JavaScript (jQuery), C#, .NET, SQL Server, API REST.

Controle de Produção de Pedidos de Restaurantes

No cenário dinâmico e acelerado dos restaurantes, a otimização dos processos é fundamental para garantir a satisfação dos clientes e a operação tranquila. Em um ambiente onde cada segundo conta, a gestão dos pedidos, desde o momento em que são feitos até chegarem à mesa dos clientes, é uma tarefa complexa.

Para resolver esse desafio criei uma interface kanban, oferecendo uma maneira visual e organizada de rastrear, gerenciar e aprimorar o fluxo de produção dos pedidos, aumentando em aproximadamente 20% a produtividade na entregra dos pedidos.

Tecnologias utilizadas: HTML, CSS, JavaScript (jQuery), .NET, C#, SQL Server.
